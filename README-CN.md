## 安装

通过 .Net CLI 工具来安装

    dotnet add package Tea

通过 NuGet 程序包管理器来安装：

* 在 `解决方案资源管理器面板` 中右击您的项目选择 `管理 NuGet 程序包` 菜单，在打开的 `NuGet 管理面板` 中点击 `浏览` 选项卡输入 `Tea`，在下方列表中选择 `Authors` 为 `Alibaba Cloud` 由官方发布的包点击 **安装** 即可。
